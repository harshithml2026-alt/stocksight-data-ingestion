ITEM 5.05. AMENDMENTS TO THE REGISTRANT’S CODE OF ETHICS, OR WAIVER OF A PROVISION OF THE CODE OF ETHICS.

On July 20, 2006, the Board of Directors of Amazon.com, Inc. (the “Company”) amended the Company’s Code of Business Conduct and Ethics (the “Code”). The amendments revise the complaint reporting provisions of the Code to allow for separate reporting guidelines where necessary to comply with the laws of other countries.

A copy of the amended Code is attached as Exhibit 14.1 to this Current Report on Form 8-K and is incorporated by reference herein.
